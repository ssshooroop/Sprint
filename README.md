# Sprint Runner - Нативное Android приложение для Wear OS

Профессиональное нативное приложение для Wear OS, разработанное на Kotlin с использованием современных технологий Android development. Приложение предназначено для тренировок спринтов с точным таймером, аналитикой прогресса и системой достижений.

## Возможности

### 🏃‍♂️ Тренировки
- **Два режима**: По времени и по дистанции
- **Точный таймер**: С точностью до сотых долей секунды
- **Интервальные тренировки**: Настройка циклов бег-отдых
- **GPS отслеживание**: Для тренировок по дистанции

### 📊 Аналитика
- **История тренировок**: Полный список с возможностью удаления
- **Статистика**: Лучшее время, среднее время, количество тренировок
- **Графики прогресса**: Визуализация улучшения результатов
- **Система достижений**: Мотивация через награды

### ⌚ Оптимизация для часов
- **Крупные элементы**: Легко нажимать пальцем
- **Экономия батареи**: Темная тема и оптимизированная графика
- **Быстрая навигация**: Swipe-жесты для переключения экранов
- **Адаптивный дизайн**: Поддержка всех размеров экранов Wear OS

## Технологический стек

## Установка и сборка

### Требования
- Android Studio Arctic Fox (2020.3.1) или новее
- Kotlin 1.8.10+
- Gradle 8.1.2+
- Android SDK 34

### Шаги по сборке

1. **Клонирование репозитория**
   ```bash
   git clone https://github.com/yourusername/sprint-runner-wear.git
   cd sprint-runner-wear
   ```

2. **Открытие в Android Studio**
   - Откройте Android Studio
   - Выберите "Open an existing project"
   - Укажите путь к папке проекта

3. **Синхронизация проекта**
   - Android Studio автоматически предложит синхронизировать проект
   - Нажмите "Sync Now" в верхней панели
   - Дождитесь завершения синхронизации

4. **Настройка устройства для разработки**
   - Подключите Wear OS устройство по USB или настройте эмулятор
   - Включите режим разработчика на устройстве
   - Разрешите отладку по USB

5. **Сборка и установка**
   ```bash
   # Сборка debug версии
   ./gradlew assembleDebug
   
   # Установка на устройство
   ./gradlew installDebug
   ```

### Настройка эмулятора Wear OS

1. Откройте Android Studio
2. Перейдите в Device Manager
3. Нажмите "Create device"
4. Выберите категорию "Wear OS"
5. Выберите нужный размер экрана (Small Round, Large Round)
6. Скачайте системный образ и создайте эмулятор

## Использование приложения

### Запуск тренировки
1. Откройте приложение на часах
2. Выберите режим тренировки (по времени или дистанции)
3. Настройте параметры при необходимости
4. Нажмите кнопку "СТАРТ"
5. Следуйте инструкциям на экране

### Просмотр результатов
- **История**: Свайп вправо от главного экрана
- **Прогресс**: Свайп влево от главного экрана
- **Детали**: Нажмите на карточку тренировки в истории

### Навигация
- **Swipe Up**: Настройки тренировки
- **Swipe Down**: Вернуться назад
- **Swipe Left/Right**: Переключение между основными экранами

# Дизайн-план Sprint Runner для Wear OS

## Философия дизайна

### Принципы дизайна для часов
- **Крупные элементы**: Все интерактивные элементы должны быть легко нажимаемыми пальцем
- **Минимализм**: Максимум одно действие на экране
- **Читаемость**: Крупный шрифт и хороший контраст
- **Быстрый доступ**: Все основные функции доступны за 1-2 нажатия
- **Экономия батареи**: Темная тема и оптимизированная графика

## Цветовая палитра

### Основные цвета
- **Primary**: #1a73e8 (глубокий синий для активных элементов)
- **Secondary**: #34a853 (зеленый для успешных действий)
- **Error**: #ea4335 (красный для стопа и ошибок)
- **Background**: #000000 (черный фон для экономии батареи)
- **Surface**: #1f1f1f (темно-серый для карточек)
- **Text Primary**: #ffffff (белый для основного текста)
- **Text Secondary**: #b0b0b0 (светло-серый для вторичного текста)

### Цвета состояний
- **Preparation**: #ff9800 (оранжевый для фазы подготовки)
- **Active**: #4caf50 (зеленый для активного спринта)
- **Rest**: #2196f3 (синий для фазы отдыха)
- **Pause**: #ffeb3b (желтый для паузы)

## Типографика

### Шрифты
- **Display Font**: "Roboto" - системный шрифт Android для заголовков
- **Body Font**: "Roboto" - тот же шрифт для всего текста
- **Моноширинный**: "JetBrains Mono" для отображения времени

### Размеры шрифтов для часов
- **Timer Display**: 32sp (очень крупный для таймера)
- **Headlines**: 18sp (заголовки экранов)
- **Body Text**: 14sp (основной текст)
- **Caption**: 12sp (вспомогательный текст)
- **Button Text**: 16sp (текст на кнопках)

## Компоненты интерфейса

### 1. Timer Display
- Крупный цифровой дисплей (32sp)
- Моноширинный шрифт для равномерного отображения
- Цвет текста меняется в зависимости от фазы тренировки
- Подсветка при изменении значений

### 2. Progress Ring
- Круговой индикатор прогресса вокруг таймера
- Гладкая анимация заполнения
- Цвет кольца соответствует текущей фазе
- Толщина линии 8dp для хорошей видимости

### 3. Control Buttons
- Крупные круглые кнопки (56dp минимум)
- Яркие цвета для различных действий
- Иконки Material Design
- Визуальная обратная связь при нажатии

### 4. Cards
- Скругленные углы (16dp)
- Темный фон с легкой тенью
- Минимальное содержимое (1-2 элемента)
- Плавные анимации при наведении

## Экраны приложения

### 1. Главный экран (TimerScreen)
**Layout**:
- Timer Display в центре (40% экрана)
- Progress Ring вокруг таймера
- Control Buttons внизу экрана
- Настройки режима вверху

**Интерактивные элементы**:
- Кнопка Старт/Стоп (большая, зеленая/красная)
- Кнопка Пауза (средняя, желтая)
- Swipe вверх для настроек
- Swipe вправо для истории
- Swipe влево для прогресса

### 2. Экран настроек
**Time Mode Settings**:
- Slider для времени подготовки (5-30 сек)
- Slider для времени спринта (10-60 сек)
- Slider для количества циклов (1-10)
- Slider для времени отдыха (30-180 сек)

**Distance Mode Settings**:
- Кнопки выбора дистанции (30, 50, 100, 200, 400м)
- Slider для времени подготовки

### 3. Экран истории
- Список карточек с результатами
- Каждая карточка содержит: время, дату, тип тренировки
- Swipe для удаления записи
- Кнопка фильтрации вверху

### 4. Экран прогресса
- Круговой график (Pie Chart) с распределением по типам
- Карточки с ключевыми метриками
- Список достижений
- Кнопка экспорта данных

## Анимации и переходы

### Используемые библиотеки
- **Compose Animation**: Стандартные анимации Jetpack Compose
- **Accompanist**: Дополнительные анимации для Compose

### Типы анимаций
1. **Timer Animation**: Плавное изменение цифр
2. **Progress Ring**: Анимация заполнения кольца
3. **Screen Transitions**: Скольжение между экранами
4. **Button Press**: Масштабирование при нажатии
5. **Card Animations**: Появление/исчезновение карточек

### Параметры анимаций
- **Duration**: 300ms для UI элементов, 100ms для обратной связи
- **Easing**: FastOutSlowIn для естественного движения
- **Stagger**: 50ms задержка для списков

## Навигация

### Жесты
- **Swipe Up**: Открыть настройки
- **Swipe Down**: Вернуться к предыдущему экрану
- **Swipe Left/Right**: Переключение между основными экранами
- **Long Press**: Контекстное меню
- **Double Tap**: Быстрое действие (пауза/старт)

### Навигационная иерархия
```
Main Screen (Timer)
├── Settings Screen
│   ├── Time Mode Settings
│   └── Distance Mode Settings
├── History Screen
│   └── Workout Details
└── Progress Screen
    └── Achievement Details
```

## Адаптивность

### Размеры экранов часов
- **Small (36mm)**: Максимум 2 элемента на экране
- **Medium (40-44mm)**: Стандартный макет
- **Large (48mm+)**: Увеличенные отступы и размеры шрифтов

### Ориентация
- Всегда портретная ориентация
- Центрирование контента для круглых экранов
- Учет cutout area (черный участок сверху)

## Accessibility

### Размеры касания
- **Minimum Touch Target**: 48dp x 48dp
- **Button Size**: 56dp для основных действий
- **Spacing**: 8dp между интерактивными элементами

### Цветовой контраст
- **Text on Background**: 15:1 (WCAG AAA)
- **Text on Surface**: 12:1 (WCAG AAA)
- **Icon on Background**: 10:1 (WCAG AA)

### Дополнительные возможности
- TalkBack поддержка для всех элементов
- Вибро-отклик для важных действий
- Высокий контраст режим
- Крупный шрифт режим

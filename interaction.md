# Интерактивный план Sprint Runner

## Основные сценарии взаимодействия

### 1. Запуск тренировки по времени
**Пользовательский путь:**
1. Пользователь открывает приложение
2. Выбирает режим "По времени"
3. Настраивает параметры:
   - Время подготовки (5-30 секунд)
   - Время спринта (10-60 секунд)
   - Количество циклов (1-10)
   - Время отдыха между циклами (30-180 секунд)
4. Нажимает "Старт"
5. Происходит обратный отсчет подготовки
6. Начинается основное время спринта
7. По завершении - автоматический переход к отдыху или следующему циклу
8. Сохранение результата в историю

**Интерактивные элементы:**
- Слайдеры для настройки времени
- Анимированный круг прогресса
- Звуковые сигналы на переходах
- Визуальная индикация текущей фазы

### 2. Запуск тренировки по дистанции
**Пользовательский путь:**
1. Выбор режима "По дистанции"
2. Выбор дистанции из предустановленных (30, 50, 100, 200, 400м)
3. Настройка времени подготовки
4. Нажатие "Старт"
5. Обратный отсчет подготовки
6. Начало отслеживания времени и "условного GPS"
7. Пользователь нажимает "Стоп" при завершении дистанции
8. Сохранение результата

**Интерактивные элементы:**
- Кнопки выбора дистанции с подсветкой
- Симуляция GPS отслеживания
- Визуализация пройденного пути
- Анимация завершения

### 3. Просмотр истории
**Функциональность:**
- Список всех тренировок с датой и результатом
- Фильтрация по типу (время/дистанция)
- Фильтрация по дате (неделя/месяц/все время)
- Удаление отдельных записей свайпом
- Просмотр деталей тренировки по клику

**Интерактивные элементы:**
- Pull-to-refresh для обновления списка
- Анимированное удаление карточек
- Всплывающие меню действий
- Плавная прокрутка

### 4. Анализ прогресса
**Функциональность:**
- Интерактивный график времени спринтов
- Переключение периодов (неделя/месяц/3 месяца/год)
- Фильтрация по дистанциям
- Показ статистики:
  - Лучшее время за период
  - Среднее время
  - Количество тренировок
  - Тренд улучшения

**Интерактивные элементы:**
- Наведение на точки графика для деталей
- Анимированное переключение периодов
- Интерактивная легенда графика
- Бейджи достижений с анимацией получения

## Состояния компонентов

### Таймер
- **Ожидание**: Серый цвет, кнопка "Старт"
- **Подготовка**: Оранжевый цвет, обратный отсчет
- **Активный спринт**: Зеленый цвет, идет отсчет времени
- **Отдых**: Синий цвет, пауза между циклами
- **Завершено**: Зеленый цвет, показ результата

### Кнопки управления
- **Старт**: Зеленая кнопка с иконкой play
- **Пауза**: Желтая кнопка с иконкой pause
- **Стоп**: Красная кнопка с иконкой stop
- **Сброс**: Серая кнопка для возврата к настройкам

### Навигация
- **Главная**: Иконка дома
- **История**: Иконка списка
- **Прогресс**: Иконка графика
- Активная вкладка подсвечивается основным цветом

## Обратная связь пользователю

### Визуальная
- Анимации при нажатии кнопок
- Изменение цветов для индикации состояния
- Прогресс-бары для наглядности
- Всплывающие уведомления о действиях

### Аудио (опционально)
- Звук старта тренировки
- Сигнал перехода между фазами
- Звук завершения тренировки
- Можно отключить в настройках

### Тактиальная (на мобильных)
- Вибрация при нажатии критически важных кнопок
- Вибрация при переходах между фазами тренировки

## Обработка ошибок
- Проверка корректности введенных параметров
- Уведомления о невозможности сохранения данных
- Информация о необходимости разрешений (для GPS)
- Грациозная деградация при отсутствии интернета

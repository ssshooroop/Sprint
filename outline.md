# Sprint Runner - Нативное Android приложение для Wear OS

## Общее описание
Нативное приложение для Wear OS на Kotlin с использованием Jetpack Compose и Material Design 3. Оптимизировано для маленьких экранов часов с крупными кнопками и простой навигацией.

## Основные компоненты

### 1. MainActivity
- Точка входа в приложение
- Настройка Wear OS UI
- Инициализация навигации

### 2. TimerScreen (Главный экран)
- Крупный дисплей таймера
- Круг прогресса
- Кнопки управления (Старт/Стоп/Пауза)
- Настройки режимов тренировки

### 3. HistoryScreen
- Список сохраненных результатов
- Возможность удаления записей
- Фильтрация по типу тренировки

### 4. ProgressScreen
- График прогресса (используя Compose Charts)
- Статистика результатов
- Система достижений

### 5. Navigation
- Swipe-based навигация для часов
- Переходы между экранами
- Сохранение состояния

### 6. Database (Room)
- Локальное хранение результатов
- Эффективные запросы для статистики
- Миграции схемы

## Технические особенности для Wear OS

### UI/UX
- Крупные кнопки для сенсорного управления
- Простая навигация свайпами
- Оптимизация для круглых экранов
- Экономия батареи

### Функциональность
- Фоновая работа таймера
- Сохранение состояния при переключении экранов
- Поддержка различных размеров экранов
- Интеграция с системными службами Wear OS

### Архитектура
- MVVM паттерн
- Dependency Injection (Hilt)
- Coroutines для асинхронных операций
- Flow для реактивного программирования

## Особенности реализации

### Таймер
- Точность до сотых долей секунды
- Обработка пауз и возобновления
- Фоновая работа с уведомлениями
- Звуковые и вибро-сигналы

### Режимы тренировки
- **Time Mode**: Интервальные тренировки с настройкой циклов
- **Distance Mode**: Тренировки на заданную дистанцию
- Автоматическое переключение между фазами
- Настройка времени подготовки и отдыха

### Аналитика
- Расчет статистики (лучшее/среднее время)
- Построение графиков прогресса
- Система достижений
- Экспорт данных

## Интеграция с Wear OS

### Tiles
- Быстрый доступ к таймеру
- Отображение последнего результата
- Ярлык для быстрого запуска

### Complications
- Мини-табло с информацией о тренировках
- Быстрый запуск приложения
- Отображение статуса тренировки

### Health Services
- Интеграция с Google Fit
- Отслеживание активности
- Синхронизация данных о тренировках
